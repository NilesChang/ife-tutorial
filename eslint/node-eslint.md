# eslint-config-node

Node Style Guide for Node.

## Init

Using `eslint --init` to generate the `.eslintrc` file.

## Usage

package.json

```json
{
  "devDependencies": {
    "eslint-config-egg": "~1.0.0",
    "eslint": "1"
  }
}
```

Edit `.eslintrc`

```json
{
  "extends": "eslint-config-egg"
}
```
