# eslint-config-react

Node Style Guide for React.

## Usage

package.json

```json
{
  "devDependencies": {
    "eslint-config-airbnb": "^9.0.1",
    "eslint-plugin-import": "^1.12.0",
    "eslint-plugin-react": "^5.2.2",
    "eslint-plugin-jsx-a11y": "^2.0.1",
    "eslint": "^3.2.0"
  }
}
```

Edit `.eslintrc`

```json
{
  "extends": "airbnb"
}
```
